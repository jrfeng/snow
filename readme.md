Android music player framework. Compatible with MediaSession.

**Support:**

* Custom music player (MediaPlayer, ExoPlayer)
* Custom Notification
* Custom audio effect engine
* Only WiFi network
* Sound quality/dynamic URL
* Headset clicks
* Sleep timer
* Playback history
* Player state persistence
